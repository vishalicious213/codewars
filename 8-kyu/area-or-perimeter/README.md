<h2><a href=https://www.codewars.com/kata/5ab6538b379d20ad880000ab/train/javascript target="_blank">Area or Perimeter</a></h2><h3>8 kyu</h3><p>You are given the <code>length</code> and <code>width</code> of a 4-sided polygon. The polygon can either be a rectangle or a square.<br>If it is a square, return its area. If it is a rectangle, return its perimeter.</p><p><strong>Example(Input1, Input2 --&gt; Output):</strong></p><pre><code>6, 10 --&gt; 323, 3 --&gt; 9</code></pre><p><strong>Note:</strong> for the purposes of this kata you will assume that it is a square if its <code>length</code> and <code>width</code> are equal, otherwise it is a rectangle.</p>
