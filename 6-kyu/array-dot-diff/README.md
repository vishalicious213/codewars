<h2><a href=https://www.codewars.com/kata/523f5d21c841566fde000009/train/javascript target="_blank">Array.diff</a></h2><h3>6 kyu</h3><p>Implement a function that computes the difference between two lists. The function should remove all occurrences of elements from the first list (<code>a</code>) that are present in the second list (<code>b</code>). The order of elements in the first list should be preserved in the result.</p><h2 id="examples">Examples</h2><p>If <code>a = [1, 2]</code> and <code>b = [1]</code>, the result should be <code>[2]</code>.</p><p>If <code>a = [1, 2, 2, 2, 3]</code> and <code>b = [2]</code>, the result should be <code>[1, 3]</code>.</p>
