Some customers try to cheat the system by entering invalid codes or using expired coupons.</p><h2 id="task">Task</h2><p>Your mission:<br>Write a function called <code>checkCoupon</code> which verifies that a coupon code is valid and not expired.</p><ul><li>A coupon is no more valid on the day <strong>AFTER</strong> the expiration date.</li><li>All dates will be passed as strings in this format: <code>"MONTH DATE, YEAR"</code>.</li><li>For the correct code and the entered code to match, both their values and data types must be the same.
